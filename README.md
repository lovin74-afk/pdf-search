# PDF Searcher

여러 PDF를 읽어 색인을 만들고, 검색어가 들어 있는 파일과 위치를 찾아주는 로컬 앱입니다.

## 기능

- 폴더 단위로 여러 PDF를 재귀적으로 색인
- 여러 PDF 업로드 후 즉시 메모리 색인
- 검색 결과에 파일명, 페이지, 줄 번호, 좌표 근사값, 문맥 표시
- 색인을 로컬 SQLite DB `.pdf_search_index/indexes.db`에 저장하고 다시 불러오기
- 앱 시작 시 마지막으로 사용한 색인을 자동으로 불러오기
- 검색 결과 수 제한 없이 전체 결과 표시

## 실행

```powershell
streamlit run app.py
```

브라우저가 열리면 왼쪽 사이드바에서 PDF 폴더 경로를 입력하거나 PDF를 업로드해 색인을 만든 뒤 검색어를 입력하면 됩니다.

색인을 만들면 자동으로 로컬 DB에 저장되며, 사이드바에서 저장된 색인을 다시 선택해 불러올 수 있습니다.

## 참고

- 좌표와 줄 번호는 PDF 텍스트 조각의 배치를 바탕으로 계산한 근사값입니다.
- 스캔 이미지 기반 PDF는 텍스트 레이어가 없으면 검색되지 않습니다.
